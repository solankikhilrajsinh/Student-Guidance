
## Student Guidance App

The Student Guidance App is a comprehensive mobile application developed using Kotlin, Android development, and Firebase. It is designed to assist students and visitors at VIT Bhopal by providing detailed faculty information, campus navigation, and a range of interactive features. The app integrates Google Maps for real-time navigation and includes a modern, user-friendly interface built with strong UI/UX principles. The project demonstrates full-stack mobile development, covering both frontend and backend implementation.


## Features
- Faculty Directory: View faculty details including name, cabin number, department, email, and other essential information.
- Campus Navigation: Real-time navigation between any two locations within the VIT Bhopal campus using Google Maps integration.
- Search Functionality: Built-in SearchView for quick filtering of faculty and campus locations.
- Voice Control: Voice-enabled search for improved accessibility and convenience.
- Google Maps Integration: Seamless map interface for accurate route guidance across the campus.
- Contact Us Page: Direct access to support and important contact information.
- Navigation Drawer: Easy access to multiple app sections with a clean navigation menu.
- Analog Clock Component: Built-in clock widget for enhanced visual design.
- Collapsing Toolbar & Custom UI: Modern UI elements including a collapsing toolbar, custom status bar, custom action bar, and GIF integration for a polished user experience.
- Permission Handling: Secure handling of app permissions for location and other features.

## Tech Stack
- Kotlin (Android Development)
- Firebase (Database, Authentication, Storage)
- Google Maps API
- Android XML UI
- Material Design Components


## How It Works
1. Users can browse or search faculty details through the interactive directory.
2. The app retrieves information from Firebase, ensuring data accuracy.
3. Using Google Maps, the user can select start and end points for campus navigation.
4. Additional interactive elements like voice control, navigation drawer, and custom UI enhance usability.
